## What it does

Dear Applicant is an open-source job application tracker built for the modern job hunt. Applications live on a drag-and-drop kanban board — move them through stages as you progress. A companion browser extension scrapes job postings directly from listings and autofills your application details, cutting out the copy-paste.

## How it's built

Fully serverless on Cloudflare's stack: Pages for hosting, D1 (SQLite at the edge) for the database, and Drizzle ORM for type-safe queries. Auth is handled by Better Auth. The frontend uses React Router v7 with TypeScript throughout.
